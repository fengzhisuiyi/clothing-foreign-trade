# 服装外贸

* 方式一（最理想）：中介外贸公司

1. 有中介外贸公司的情况下，联系中介外贸公司，把样板寄过去，中介外贸公司一般有自己的客户源，可以快速进行沟通、调整、铺货。

2. 让专业的服装销售员对接需求。

3. 有需求后，对服装进行调整确认。

4. 批量下单。

   

* 方式二：国外线下服装批发市场展示样板

1. 线下找国外的人口多的城市的服装批发市场，把样板找摊位展示出来，看有没有客户对接。
2. 有客户的情况下，让专业的服装销售员对接需求。
3. 有需求后，对服装进行调整确认。
4. 批量下单。
